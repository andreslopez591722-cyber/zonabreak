📧 Reporte Diario por Correo
⚡ El sistema puede enviarte un resumen del día automáticamente por correo.
Usa EmailJS (gratis hasta 200 correos/mes). Configura una vez y listo.
⚙️ Configuración EmailJS
Pasos para configurar:
1. Crea cuenta gratuita en emailjs.com
2. Crea un "Email Service" (Gmail, Outlook, etc.)
3. Crea un "Email Template" con variables: {{reporte}}
4. Copia el Service ID, Template ID y Public Key
5. Pégalos aquí y guarda ✅
📅 Envío Automático
